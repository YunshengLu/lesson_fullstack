- 为何要优化函数？
    1. 函数式组件               
        还会定义函数，闭包  函数里可以定义函数              
        组件更新的时候，函数是不是会再次定义？
    2. useCallback 来缓存函数 useMemo缓存函数计算的结果               
    3. 函数要传给子组件               
        useCallback(要缓存的函数，[依赖])              
        useMemo 当组件里的状态发生改变的时候，不管与计算结果的函数是否相关                
        都会重新计算 结果都是一样的，没必要，需要缓存计算结果
